SKYSCRAPERS PROJECT


About the program

The skyscrapers program is used for validating if the board with an input combination of skyscrapers contains a winning
combination or not. The houses of different hights are places on the board, and the amount of houses seen must be equal
to the hint given. None of the houses in the column or row are of the same height.
If this rules are satisfied, the combination is winning.


What this program does?

This program checks unique houses in the column and the row. It also checks if the amount of houses seen is equal to the
hint given. If all checks return the True result, the program outputs the True result, meaning that the combination is winning.
Else the program outputs the False result.


Instruction on how to use the program

Change the name of the file, which contains the combination of skyscrapers on the board. Start the program and wait for
the result.


Results of the work

A program that validates if the combination of skyscrapers on the board is winning or not.
